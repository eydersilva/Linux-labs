# Gerenciamento de usuários no Linux

## Comandos

- whoami → mostra o usuário atual.
- id → exibe informações do usuário.
- sudo → executa comandos como administrador.
- passwd → altera a senha do usuário.
- adduser → cria um novo usuário.
- userdel → remove um usuário.
- groups → mostra os grupos do usuário.
